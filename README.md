## 🚀 Featured Projects

<details>
<summary><b>🚗 Face Controlled Car</b></summary>

<br/>

> An AI-powered system that controls a physical car using real-time facial gestures and head movement detection — no hands required.

| Attribute | Details |
|-----------|---------|
| **Stack** | Python · OpenCV · MediaPipe · Arduino · Raspberry Pi |
| **Scale** | Real-time processing at 30fps |
| **Performance** | <50ms latency from gesture to motor response |
| **Security** | Face authentication before control activation |
| **Impact** | Assistive technology for differently-abled individuals |
| **Repository** | [![GitHub](https://img.shields.io/badge/View%20Repo-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/Shreyansh-Sri) |

This project uses **MediaPipe Face Mesh** to track 468 facial landmarks in real time. Head tilt direction is mapped to motor control signals sent via serial communication to an Arduino, which drives the car's motor driver. The system includes a face verification layer ensuring only the authenticated user can operate the vehicle.

</details>

<details>
<summary><b>🎓 AI Attendance System</b></summary>

<br/>

> An automated attendance management system using real-time face recognition — eliminating manual roll calls and proxy attendance entirely.

| Attribute | Details |
|-----------|---------|
| **Stack** | Python · OpenCV · face_recognition · Flask · SQLite |
| **Scale** | Handles 100+ student profiles per classroom |
| **Performance** | 98%+ face recognition accuracy under standard lighting |
| **Security** | Anti-spoofing detection, encrypted attendance logs |
| **Impact** | Reduces attendance marking time from 10 mins to under 30 seconds |
| **Repository** | [![GitHub](https://img.shields.io/badge/View%20Repo-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/Shreyansh-Sri) |

The system captures live video from a classroom camera, detects and encodes faces using **dlib-based face embeddings**, and matches them against a pre-registered student database. Attendance is logged automatically into SQLite with timestamps, and a Flask dashboard allows faculty to export reports instantly.

</details>
